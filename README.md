## [cocos-playable-demo](https://github.com/magician-f/cocos-super-html)

基于cocos creator，对接一次，即可支持众多试玩平台

## 试玩平台介绍

- 关于交付文件，根据不同试玩平台的要求，最终只能交付1个文件，比如：index.html 或 index.zip
